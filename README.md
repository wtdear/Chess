# ♟️ SharpChess

A lightweight, indie Chess engine built with **C#**. This project focuses on clean object-oriented design and efficient move validation.

## ✨ Key Features
* **Full Rule Support:** Includes castling, en passant, and pawn promotion.
* **C# Logic:** Pure OOP approach for board representation and piece behavior.
* **Extensible:** Designed to easily integrate a custom UI or an AI opponent.
* **Lightweight:** No heavy dependencies, just clean .NET code.
